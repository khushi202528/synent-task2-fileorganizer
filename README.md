# File Organizer (Python)

This project organizes files in a folder based on their file type.

## Features
- Automatically sorts files into folders
- Categories:
  - Images
  - Documents
  - Videos
  - Others
- Handles multiple file extensions
- Creates folders automatically

## Technologies Used
- Python
- os module
- shutil module

## How to Run
1. Run the Python file
2. Enter folder path
3. Files will be automatically organized
